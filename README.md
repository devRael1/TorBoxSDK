# TorBoxSDK

An open-source C# SDK for the [TorBox API](https://api-docs.torbox.app/) — compatible with .NET 6 through .NET 10.

## APIs Covered

| API | Base URL | Status |
|-----|----------|--------|
| Main API | `https://api.torbox.app/v1/api` | Planned |
| Search API | `https://search-api.torbox.app` | Planned |
| Relay API | `https://relay.torbox.app` | Planned |
